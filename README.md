# Nifty-100-Financial-Intelligence-Platform
📈 A comprehensive financial analytics platform for Nifty 100 companies built using Streamlit, SQLite, Pandas, and Plotly. The application provides interactive dashboards, financial data management, company analysis, and business intelligence visualizations.

## 🚀 Features

- Financial data management for Nifty 100 companies
- Relational database storage using SQLite
- Interactive dashboards and charts
- Company-wise financial analysis
- Revenue and profit tracking
- Sector-wise comparison
- Financial performance visualization
- Data upload and processing capabilities
- Dynamic business intelligence reports

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python
- **Database:** SQLite
- **Data Processing:** Pandas, NumPy
- **Visualization:** Plotly
- **Deployment:** Streamlit + LocalTunnel
